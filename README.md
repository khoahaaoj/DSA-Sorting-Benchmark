# DSA-Sorting-Benchmark

## Giới thiệu
Dự án này thực hiện cài đặt và so sánh thời gian thực thi của các thuật toán sắp xếp trên tập dữ liệu lớn (1 triệu phần tử/dãy).

## Các thuật toán được kiểm thử
1. **QuickSort** 
2. **MergeSort**
3. **HeapSort**
4. **NumPy Sort** 

## Dữ liệu thử nghiệm
- File: `dataset.zip` (Vui lòng giải nén để lấy file `dataset.csv`)
- Kích thước: 10 dãy số, mỗi dãy 1.000.000 phần tử.
- Bao gồm: Số nguyên (Int) và Số thực (Float) với các trật tự Tăng dần, Giảm dần và Ngẫu nhiên.
